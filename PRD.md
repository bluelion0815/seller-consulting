# PRD — SellerPilot (셀러 컨설팅 서비스)

> 과제 1번 산출물. "같은 흐름을 내 주제로 완주" — 멀티마켓 셀러 컨설팅 랜딩.

## 1. 목적
스마트스토어·롯데온·쿠팡 등 **여러 마켓을 운영하는 셀러**에게
진단→세팅→운영대행→교육 컨설팅을 소개하고 **무료 진단 상담 신청**으로 전환.

## 2. 타깃 / 핵심 메시지
- 타깃: 매출 정체·운영 과부하를 겪는 1인~중소 셀러
- 한 줄 가치: **"혼자 운영하지 마세요 — 멀티마켓 매출, 전문가와 함께"**
- 톤: 신뢰·전문성, 에메랄드 포인트 + 다크모드

## 3. 페이지 정의 (4p)
| 경로        | 페이지   | 목표        | 핵심 섹션 |
|-------------|----------|-------------|-----------|
| `/`         | 홈       | 신뢰·전환   | Hero, 문제공감, 서비스 4종, 성과지표, 후기, CTA |
| `/services` | 서비스   | 이해        | 진단·채널세팅·운영대행·교육 상세 |
| `/pricing`  | 요금     | 비교        | 3-tier 패키지 + FAQ |
| `/contact`  | 상담신청 | 전환        | 무료 진단 신청 폼(더미) |
| `/styleguide` | (내부) | 디자인 SSOT | 토큰·컴포넌트 |

## 4. 성공 기준
- [ ] 4개 페이지 라우팅 + 네비 이동
- [ ] 반응형 + 다크모드
- [ ] 공용 컴포넌트만 사용
- [ ] `next build` 통과
- [ ] **Vercel URL 도출 → 1번 과제 제출**

## 5. 범위 밖
실제 결제·CRM·DB 연동 → 전부 더미. 실제 고객데이터 미사용.
